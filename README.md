# Ventaj — элитный сайт винтажной одежды

Этот проект — статический одностраничный сайт для продажи дорогих винтажных вещей. Он сочетают в себе современный дизайн, эмоциональную презентацию коллекций и мощную аналитику рынка.

## Что сделано

- Современный адаптивный лендинг с тёмной премиальной эстетикой.
- Разделы: дропы, преимущества, аналитика рынка, история бренда, CTA.
- Контент, который вызывает желание купить редкую вещь прямо сейчас.
- Минималистичный, не перегруженный дизайн, но с высокой информативностью.

## Анализ рынка и конкуренты

### Тренды
- Рынок ресейла роскоши растёт более чем на 20% в год.
- Дорогой винтаж становится инструментом самовыражения для молодых коллекционеров и артистов.
- Спрос на эксклюзивные вещи сильнее, чем когда-либо: уникальность, история и гарантия подлинности решают.

### Основные конкуренты / аналоги
- `Vestiaire Collective` — крупная платформа для ресейла премиальных вещей.
- `TheRealReal` — лидер в сегменте проверенного люкса с большим каталогом.
- `Grailed` — сообщество стритвира и редких коллекционных вещей.
- `Farfetch` — премиум-платформа с винтажными и архивными изделиями.

### Наше отличие
- Фокус на эмоциональной презентации и редких лотах.
- Подача как коллекция культурных объектов, а не просто товаров.
- Лёгкая навигация и мощный визуальный стиль, который работает на конверсию.

## Запуск
Откройте `index.html` в браузере.
